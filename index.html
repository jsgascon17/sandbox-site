CSS Animations

Learn how to make elements move, fade, and transform

1. Fade In

Makes an element appear by changing its opacity from 0 to 1.

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.element {
  animation: fadeIn 2s ease-in-out;
}

2. Slide In

Makes an element slide in from the side while fading in.

@keyframes slideIn {
  from { opacity: 0; transform: translateX(-100px); }
  to { opacity: 1; transform: translateX(0); }
}

3. Bounce

Makes an element bounce up and down continuously.

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-30px); }
}

4. Spin

Rotates an element 360 degrees continuously.

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

5. Pulse

Makes an element grow and glow, then shrink back.

@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

6. Hover Effects (no @keyframes needed!)

Hover over this box. Uses transition instead of animation.

.element {
  transition: all 0.3s ease;
}

.element:hover {
  transform: scale(1.2) rotate(5deg);
}

7. Combined Animation + Hover

Float animation + hover glow effect together.

@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-15px); }
}

.element {
  animation: float 4s ease-in-out infinite;
  transition: box-shadow 0.3s ease;
}

.element:hover {
  box-shadow: 0 30px 60px rgba(168, 85, 247, 0.4);
}

Key Concepts

@keyframes - Defines the animation steps (from/to or percentages)

animation - Applies the keyframe to an element (name, duration, timing, iterations)

transition - Smooth change on state change (like hover) - simpler than keyframes

transform - Move (translate), rotate, scale, or skew elements

infinite - Makes the animation loop forever

alternate - Plays the animation forward, then backward
